In deze les heb ik gewerkt aan mijn projectdocumentatie voor het project "Lost in tra(i)nslation".
Eerst heb ik een GitHub-account gekoppeld en daarna dit account gelinkt aan mijn Netlify, zodat ik deze website makkelijk kan publiceren. Voor typografie heb ik gebruik gemaakt van Google Fonts. In index.html heb ik mijn basis geschreven om vervolgens de pagina te dupliceren en te hernoemen naar week 1.html, week 2.html, ... Als dit allemaal klaar was, heb ik in GitHub een nieuwe repository gemaakt en deze via Netlify gepubliceerd. Tijdens deze les hebben we ook aparte lessen gekregen over design en coderen. We leerden ook hoe we windtail moesten installeren, dat kun je hieronder in bijlage zien.
